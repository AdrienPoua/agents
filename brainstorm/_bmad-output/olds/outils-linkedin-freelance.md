Tu es un expert LinkedIn pour freelances tech. Je suis développeur
web full stack spécialisé dans les sites et apps pour clubs sportifs.

Mon contexte :
- J'ai créé 2 apps pour des clubs de basket (Argenteuil, Sartrouville)
- J'automatise la récupération des matchs via l'extranet fédéral
- Mon offre : Site club sportif 1200€ / App sur mesure 2300€

  galèrent avec des sites pourris et perdent du temps en gestion manuelle
- Mon parcours : Licence STAPS + 7 ans dans le basket + dev web

Génère-moi un post LinkedIn de 150-200 mots qui :
- Accroche dès la première ligne (pattern interrupt, chiffre choc,
  question provocante ou histoire)
- Utilise des phrases courtes (1 idée par ligne)
- Raconte une histoire OU partage un apprentissage concret OU
  donne des conseils actionnables
- Montre mon expertise sans être vendeur ou promotionnel
- Termine par une question ouverte ou un call-to-action soft
- Utilise des sauts de ligne pour la lisibilité (1 phrase = 1 ligne)
- Pas de hashtags dans le corps du texte (juste à la fin, max 3-5)

Thème du post : [METTRE TON THÈME ICI]

Exemples de thèmes :
- Avant/après d'un site de club que j'ai refait
- Une erreur commune sur les sites de clubs
- Comment j'ai automatisé les résultats de matchs
- Mon parcours atypique STAPS → dev
- Pourquoi les clubs sportifs méritent mieux qu'un site WordPress gratuit
```

La solution ? Automatiser.

J'ai créé une app pour un club de basket qui récupère
les résultats automatiquement depuis la fédé.

Plus besoin d'y penser.

Votre site de club est à jour ?

#clubsportif #sportamateur #developpeurweb
```

### Post #3 : Le Conseil Actionnable

```
3 trucs que je vérifie sur TOUS les sites de clubs
sportifs (et que vous pouvez corriger aujourd'hui) :

1. Le site est-il lisible sur mobile ?
→ 70% de vos visiteurs sont sur téléphone
→ Testez : prenez votre tel, ouvrez votre site, zoomez-vous ?

2. Les infos de contact sont-elles visibles ?
→ En moins de 5 secondes, je dois pouvoir vous contacter
→ Numéro + email + adresse = en haut de page ou menu

3. La page d'accueil dit-elle ce que vous faites ?
→ "Club de [sport] à [ville]" doit être visible immédiatement
→ Pas de slider qui prend 10 secondes à charger

Ces 3 points = 80% de l'efficacité d'un site de club.

Le reste c'est du bonus.

Votre site passe le test ?

#siteweb #clubsportif #conseils
```

```

dirigeant de club.

Et c'est VRAI.

Entre les entraînements, les matchs, la gestion,
les licences, les subventions...

Qui a le temps de :
- Publier les résultats du week-end
- Mettre à jour le calendrier
- Ajouter les photos

Personne.

Donc le site reste figé.
Et donne une image de club "mort".

Ma solution : automatiser ce qui peut l'être.

Exemple concret :
→ Connexion à l'extranet fédération
→ Les matchs et résultats arrivent tout seuls
→ 0 action manuelle

Le temps récupéré ? Pour le club. Pas pour le site.

C'est quoi votre plus grosse perte de temps au club ?

#automatisation #clubsportif #gagnezdutemps
```
