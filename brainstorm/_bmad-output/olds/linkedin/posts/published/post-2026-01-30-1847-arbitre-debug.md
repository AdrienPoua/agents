## Post (Copier-coller dans Buffer)

Arbitre national : 0.5 seconde pour décider.

Développeur : 2h pour trouver un bug.

Les deux m'ont appris la même chose.

𝗘𝗻 𝗮𝗿𝗯𝗶𝘁𝗿𝗮𝗴𝗲 :

⚡ Tu dois trancher
→ Pas le temps de douter
→ Tu assumes ta décision

𝗘𝗻 𝗱𝗲𝗯𝘂𝗴𝗴𝗶𝗻𝗴 :

🧠 Tu dois analyser
→ Reproduire le problème
→ Isoler la cause

Deux approches opposées.

Deux compétences complémentaires.

L'arbitrage m'a appris à 𝗱𝗲́𝗰𝗶𝗱𝗲𝗿 𝘃𝗶𝘁𝗲.

Le code m'a appris à 𝗽𝗿𝗲𝗻𝗱𝗿𝗲 𝗹𝗲 𝘁𝗲𝗺𝗽𝘀.

Aujourd'hui je sais quand utiliser chaque mode.

Réunion client → mode arbitre.

Bug en prod → mode debugger.

—

Quel métier inattendu t'a préparé au dev ?
