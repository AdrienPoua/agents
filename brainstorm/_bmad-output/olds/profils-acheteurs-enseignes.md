## Guide : Comprendre l'Organisation Achats

### Structure type d'une direction achats en GSS/GSA

```
Direction Generale
    |
    +-- Direction Commerciale & Marketing
    |       |
    |       +-- Directeur de l'Offre / Directeur Achats
    |       |       |
    |       |       +-- Category Managers (par univers produit)
    |       |       |       |
    |       |       |       +-- Acheteurs / Chefs de Produit (par sous-categorie)
    |       |       |       |       |
    |       |       |       |       +-- Assistants achats
    |       |       |       |
    |       |       |       +-- Responsables Merchandising
    |       |       |
    |       |       +-- Direction Supply Chain / Logistique
    |       |
    |       +-- Direction E-commerce & Digital
    |       |       |
    |       |       +-- Responsables e-commerce par categorie
    |       |
    |       +-- Direction Marketing & Communication
    |
    +-- Direction des Operations (magasins)
            |
            +-- Directeurs regionaux
                    |
                    +-- Directeurs de magasin
                            |
                            +-- Chefs de rayon / Responsables univers
```

**En GSS (Fnac, Boulanger, Electro Depot)** : l'organisation est centralisee. La direction achats/offre decide des references, des prix et du merchandising. Les magasins executent avec peu de marge de manoeuvre sur l'assortiment.

**En GSA (Carrefour, E.Leclerc)** : le non-alimentaire est gere par une direction dediee, souvent avec moins de priorite que l'alimentaire. Chez Leclerc, le systeme cooperatif donne plus d'autonomie aux adherents locaux.

**Chez Amazon** : pas de magasins physiques (hors quelques exceptions). Le Vendor Manager cumule les roles d'acheteur, category manager et negociateur. Tout est pilote par la data.

### Les roles cles que le KAM va rencontrer

| Role | Responsabilites | Pouvoir de decision | Frequence de contact |
|------|----------------|--------------------|--------------------|
| **Directeur Commercial / Achats** | Strategie globale, validation des partenariats majeurs, arbitrages budgetaires | Decisionnaire final sur les accords-cadres et les gros budgets | 1 a 2 fois/an (revues strategiques) |
| **Category Manager** | Pilotage d'un univers produit (ex: mobilite, PEM, audio). Definit l'assortiment, les objectifs de marge, le plan promo | Decide du referencement et du dereferencement. Valide les plans d'affaires | 4 a 6 fois/an (negociation + suivi) |
| **Acheteur / Chef de Produit** | Negocie les conditions commerciales, suit les stocks, gere le quotidien operationnel | Execute les decisions du CM. Negocie les details (volumes, prix, delais) | Mensuel a bi-mensuel |
| **Chef de rayon (magasin)** | Implantation en rayon, conseil client, remontee terrain sur les ventes | Influence sur la mise en avant locale. Pas de pouvoir de referencement | Selon strategie terrain |
| **Responsable e-commerce** | Contenus produits en ligne, performance digitale, SEO marketplace | Co-decide avec le CM sur la visibilite en ligne | 3 a 4 fois/an |

### Profil type d'un acheteur/category manager en GSS tech

**Parcours type :**
- Bac+5 ecole de commerce (ESSEC, EDHEC, EM Lyon, Kedge...) ou Master marketing/distribution
- Premier poste : chef de rayon ou assistant acheteur (2-3 ans)
- Evolution vers acheteur junior puis category manager en 5-8 ans
- Certains viennent du cote industriel (chef de produit chez un fabricant) et passent cote distributeur

**Age moyen et experience :**
- Category Manager : 30-40 ans, 7-12 ans d'experience
- Acheteur confirme : 28-35 ans, 4-8 ans d'experience
- Salaire CM : 50 000-70 000 EUR bruts/an + variable sur objectifs

**Ce qui les motive (objectifs et KPIs) :**
- Chiffre d'affaires de leur categorie (croissance vs N-1)
- Marge brute et marge nette
- Part de marche de l'enseigne sur leur univers produit
- Taux de rotation des stocks (eviter le surstock)
- Qualite du mix produit (innovation vs fond de rayon)
- Performance des promotions (ROI promo)

**Ce qui les stresse :**
- Ruptures de stock (le pire cauchemar -- perte de CA immediate)
- Retours clients et taux de SAV eleve
- Fournisseurs qui ne respectent pas les delais de livraison
- Pression hierarchique sur les marges
- Changements de tarifs non anticipes
- Produits qui ne tournent pas (stock mort)

**Leur journee type :**
- Matin : analyse des ventes de la veille (tableaux de bord, sell-out), emails fournisseurs
- 10h-12h : rendez-vous fournisseurs (presentations produits, negociations)
- Apres-midi : travail sur les plans d'assortiment, le merchandising, les promos a venir
- 16h-17h : reunions internes (supply chain, e-commerce, direction)
- Veille concurrentielle permanente (sites concurrents, panels GfK/Circana)

**Comment ils evaluent un fournisseur :**

| Critere | Poids | Ce qu'ils regardent |
|---------|-------|-------------------|
| Marge | Eleve | Prix d'achat net net, conditions commerciales, RFA |
| Rotation | Eleve | Historique de ventes, potentiel marche (panels) |
| Innovation | Moyen-Eleve | Differenciation vs concurrence, brevets, awards (CES, etc.) |
| Fiabilite logistique | Moyen | Taux de service, respect des delais, qualite du conditionnement |
| Support marketing | Moyen | Budget co-op, PLV, digital, formation vendeurs |
| SAV et qualite | Moyen | Taux de retour, avis clients, certification |
| Notoriete marque | Variable | Awareness consommateur, investissement media |
| Relation humaine | Sous-estime | Reactivite, transparence, respect des engagements |

## FNAC-DARTY -- Organisation Achats

### Structure achats

Fnac Darty (CA ~8 Mds EUR, ~1 000 magasins Fnac + Darty) est le leader francais de la distribution de produits techniques et culturels. La fusion Fnac-Darty (2016) a unifie les achats mais conserve deux enseignes distinctes en magasin.

| Niveau | Role | Perimetre | Infos connues (sources publiques) |
|--------|------|-----------|-----------------------------------|
| Comex | Directrice Commerciale & Marketing | Toutes categories produits | Florence Lemetais (membre du Comite Executif) |
| Comex | Directeur E-commerce & Digital | Strategie digitale, marketplace | Olivier Theulle |
| Comex | Directeur Operations | Reseau magasins | Francois Gazuit |
| Direction | Directeur Offre Services & Durabilite | Services, offre durable | Regis Koenig (nomme 2023) |
| CM | Category Managers | Par univers : Mobilite, PEM, Audio, Informatique, Photo, Gaming... | Organises par "univers produit" alignes sur les rayons |
| Acheteur | Acheteurs operationnels | Par sous-categorie au sein de chaque univers | Equipes basees au siege (Ivry-sur-Seine) |

**Point cle** : Chez Fnac Darty, la direction commerciale et marketing (Florence Lemetais) coiffe l'ensemble de l'offre. Le plan strategique "Everyday" puis "Beyond Everyday" (horizon 2030) met l'accent sur les services, la durabilite et la marketplace. Le KAM doit integrer ces priorites strategiques dans son argumentaire.

### Processus de referencement

**Etape 1 -- Premier contact**
- Identifier le Category Manager de l'univers concerne (ex: "Mobilite urbaine", "PEM cuisine", "Audio")
- Envoyer un dossier de presentation structure : fiche marque, catalogue, positionnement prix, plan marketing
- Les salons (CES, IFA) sont d'excellents points de premier contact informel

**Etape 2 -- Rendez-vous de presentation**
- Se rendre au siege (Ivry-sur-Seine ou site historique Fnac/Darty)
- Presenter : le marche (donnees GfK/Circana), la marque, les produits, l'argumentaire de differenciation
- Apporter des echantillons/demos. L'acheteur Fnac est un passione de tech -- montrer le produit en action

**Etape 3 -- Evaluation interne**
- Le CM analyse : positionnement prix vs concurrence, marge potentielle, complementarite avec l'assortiment existant
- Test possible sur un panel de magasins (10-20 magasins "pilotes")
- Analyse de la capacite logistique du fournisseur

**Etape 4 -- Negociation commerciale**
- Conditions d'achat : prix, remises, RFA (Remise de Fin d'Annee)
- Cooperation commerciale : budget promo, PLV, formation vendeurs, operations speciales
- Conditions logistiques : delais, minimum de commande, reprise des invendus

**Etape 5 -- Decision et deploiement**
- Validation par le CM et sa hierarchie
- Integration au systeme SI (creation des references)
- Deploiement progressif : e-commerce d'abord, puis magasins pilotes, puis reseau complet
- Delai global : 3 a 9 mois selon la complexite

### Calendrier de negociation

| Periode | Phase | Ce qui se passe |
|---------|-------|----------------|
| Janvier-Fevrier | Post-CES | Decouverte des nouveautes, premiers contacts sur les innovations presentees a Las Vegas |
| Mars-Mai | Planification S2 | Les CM preparent les assortiments pour la rentree et les fetes. Moment ideal pour presenter de nouveaux produits |
| Juin-Juillet | Validation S2 | Decisions d'assortiment pour la rentree. Dernieres fenetres pour le referencement Q4 |
| Septembre-Octobre | IFA + Rentree | Salon IFA (Berlin) : vitrine des nouveautes. Rentree : lancement des nouveaux produits en rayon |
| Novembre-Decembre | Black Friday + Noel | Periode de vente critique. Aucune negociation possible -- focus 100% execution |
| Septembre-Novembre | Negociation N+1 | Preparation des conditions commerciales pour l'annee suivante (en parallele de la saison haute) |

### Comment aborder un acheteur Fnac-Darty

1. **Parler le langage de la marque** : La Fnac a une culture forte de "prescripteur tech" et de "conseiller eclaire". Montrez que vos produits meritent d'etre recommandes, pas juste vendus. Les labels "Coup de Coeur Fnac" ou les tests redactionnels en magasin comptent.

2. **Arriver avec de la data** : Les equipes Fnac-Darty sont tres analytiques. Presentez des donnees panels (GfK, Circana), des tendances de marche, des comparaisons de marges. Ne venez pas avec des intuitions -- venez avec des chiffres.

3. **Proposer un plan marketing co-finance** : Fnac-Darty attend un investissement du fournisseur : operations en magasin, mise en avant sur fnac.com/darty.com, formation des vendeurs-conseils. Preparez un budget de cooperation commerciale clair.

4. **Penser omnicanal** : Le groupe pousse fortement le digital (marketplace, click & collect). Proposez des contenus produits de qualite (visuels, videos, fiches techniques enrichies) compatibles avec leur site et marketplace.

5. **Jouer la carte de la durabilite** : Le plan "Everyday" met l'accent sur la reparabilite et la durabilite. Si vos produits ont un indice de reparabilite eleve, des certifications environnementales, ou un programme SAV solide -- mettez-le en avant.

### Points de contact et salons

- **Siege social** : 9, rue des Bateaux-Lavoirs, 94200 Ivry-sur-Seine
- **CES Las Vegas** (janvier) : les equipes Fnac-Darty y envoient des delegations pour le scouting produits
- **IFA Berlin** (septembre) : rendez-vous cle pour les marques d'electronique grand public
- **Showrooms fournisseurs** : organiser des journees portes ouvertes dans vos locaux

### Structure achats

Boulanger (CA ~3,5 Mds EUR, ~200 magasins) est la 2e enseigne d'electromenager en France, filiale du groupe United.b (ex-HTM), propriete de la famille Mulliez (Association Familiale Mulliez -- aussi proprietaire de Auchan, Decathlon, Leroy Merlin...).

| Niveau | Role | Perimetre | Infos connues (sources publiques) |

| Direction Generale | Directrice Generale | Pilotage strategique | Emilie Soleri (ex-Auchan, profil digital et offre) |

| Direction | Direction E-commerce & Marketing | Ventes en ligne, marketing digital | Strategie omnicanale forte |

| Acheteur | Acheteurs / Chefs de produit | Par sous-categorie | Equipes compactes, accessibles |

**Specificites Boulanger :**
- Culture "nordiste" (siege a Lesquin, pres de Lille) : les relations sont directes, chaleureuses mais sans chichis
- Forte culture du service (installation, SAV, Boulanger Services)
- L'enseigne partage des synergies achats avec Electro Depot (meme groupe United.b)
- Emilie Soleri, la DG, a un profil "offre et digital" (ex-directrice de l'offre chez Auchan) -- l'innovation et le digital sont valorises

### Processus de referencement

**Contact initial :**
- Cibler le Category Manager du bon univers (PEM, Mobilite, Audio, Informatique)
- Boulanger recoit les fournisseurs au siege de Lesquin. Prendre rendez-vous via le standard achats
- Les salons professionnels (IFA surtout) sont un canal d'approche privilegie

**Presentation :**
- Boulanger valorise la demonstration produit. Apporter des echantillons fonctionnels
- Insister sur le SAV et la durabilite (axe strategique fort)
- Presenter le plan de formation vendeurs -- Boulanger investit massivement dans le conseil en magasin

**Decision :**
- Plus rapide que Fnac-Darty en general (structure plus legere)
- Test possible sur quelques magasins avant deploiement national
- Forte attention au rapport qualite/prix (positionnement "meilleur rapport" vs Fnac plus premium)

**Delai moyen : 2 a 6 mois**

### Calendrier de negociation

| Periode | Phase | Ce qui se passe |
|---------|-------|----------------|
| Janvier-Mars | Post-CES + Bilan N-1 | Analyse des performances, ajustement des assortiments, nouveautes CES |

### Comment aborder un acheteur Boulanger

1. **Miser sur le rapport qualite/prix** : Boulanger n'est ni low-cost ni ultra-premium. Montrez que vos produits offrent le meilleur compromis pour le consommateur.

2. **Penser "service"** : Comment votre produit s'integre dans l'ecosysteme de services Boulanger (installation, formation, SAV). Un produit facile a installer et a reparer sera privilegie.

3. **Etre concret et direct** : Culture du Nord oblige, allez droit au but. Pas de slides de 80 pages -- un pitch clair, des echantillons, des chiffres.

4. **Proposer de la formation vendeurs** : Les vendeurs Boulanger sont formes et passionnes. Offrir des sessions de formation produit est un vrai avantage competitif.

5. **Montrer la complementarite avec Electro Depot** : Si votre gamme peut s'adapter aux deux enseignes (Boulanger premium/milieu de gamme, Electro Depot entree/milieu), c'est un argument de volume.

## ELECTRO DEPOT -- Organisation Achats

### Structure achats

Electro Depot (CA ~1,2 Md EUR, ~120 magasins dont 102 en France) est le concept low-cost du groupe United.b. Positionnement "prix bas tous les jours" avec un assortiment court et percutant.

| Niveau | Role | Perimetre | Infos connues (sources publiques) |
|--------|------|-----------|-----------------------------------|
| Direction | President | Strategie globale | Stephane March |
| Direction | Directeur E-commerce & Marketing | Ventes digitales | Organisation compacte |
| CM/Acheteur | Category Manager / Acheteur | Par univers -- roles souvent fusionnes | Equipes polyvalentes au siege (Faches-Thumesnil, pres de Lille) |

**Specificites Electro Depot :**
- Assortiment tres court (vs Fnac ou Boulanger) : ~3 000 references vs ~30 000 chez Fnac
- Focus sur le prix : chaque produit doit offrir le meilleur prix du marche
- Les roles d'acheteur et de category manager sont souvent fusionnes
- Strategie MDD (Marques De Distributeur) forte -- ils developpent leurs propres marques
- Synergies achats avec Boulanger (meme groupe)
- Expansion europeenne (Belgique, Espagne)

### Processus de referencement

**Specificite cle** : Electro Depot fonctionne avec un assortiment extremement serre. Chaque nouvelle reference remplace potentiellement une existante. L'argumentaire doit donc demontrer pourquoi votre produit merite sa place PLUS qu'un concurrent deja reference.

**Etapes :**
1. Contact du Category Manager/Acheteur au siege de Faches-Thumesnil
2. Presentation centree sur le prix et la rotation : "Quel prix consommateur ? Quelle marge ? Quel volume ?"
3. Test eventuel en magasin (quelques semaines)
4. Decision rapide (l'organisation est agile)
5. Deploiement national si le test est concluant

**Delai moyen : 1 a 4 mois** (le plus rapide des enseignes cibles)

### Comment aborder un acheteur Electro Depot

1. **Le prix d'abord** : Pas de detour. L'acheteur veut savoir le prix public cible, la marge, et pourquoi c'est competitif. Point.

2. **Simplicite du produit** : Electro Depot cible le grand public. Pas de produits complexes necessitant une explication de 30 minutes. Le produit doit "se vendre seul" sur le lineaire.

3. **Volumes et rotation** : Montrez des projections de rotation. Un produit a forte marge mais faible rotation ne les interesse pas. Ils preferent la "masse de marge" (marge unitaire x volume).

4. **Packaging auto-porteur** : Peu de vendeurs en magasin (concept "depot"). Le packaging doit expliquer le produit et convaincre seul.

5. **Proposer des exclusivites** : Des references ou coloris exclusifs Electro Depot sont tres apprecies -- ca renforce leur differenciation prix.

## CARREFOUR -- Organisation Achats (Non-Alimentaire)

### Structure achats

Carrefour (CA France ~37 Mds EUR, ~5 800 magasins tous formats) est le 1er distributeur francais. Le non-alimentaire (dont l'electronique) represente une part minoritaire mais strategique du CA, surtout en hypermarches.

| Niveau | Role | Perimetre | Infos connues (sources publiques) |
|--------|------|-----------|-----------------------------------|
| Comex Groupe | Directeur Executif Marchandises | Strategie achats groupe | Herve Daudin |
| Direction France | Directrice Non-alimentaire | Textile, electromenager, multimedia, parapharmacie | Alice Rault |
| Direction | Direction Achats PEM/Multimedia | Achats electronique et electromenager | Category managers dedies |
| CM | Category Managers PEM | Petit electromenager par sous-categorie | Siege a Massy |
| Acheteur | Acheteurs operationnels | Negociation et suivi quotidien | Equipes en centrale |

**Specificites Carrefour :**
- **Centrale d'achats Envergure** : alliance Carrefour + Systeme U pour le non-alimentaire, renforce le pouvoir de negociation
- L'electronique/PEM est un univers secondaire chez Carrefour (vs alimentaire) -- budget et lineaire limites
- Offre d'emploi recente : "Category Manager Petit Electromenager" -- confirme que la categorie PEM a son propre CM
- Forte pression prix (positionnement "prix bas" renforce sous Alexandre Bompard)
- La strategie "Act for Food/Change" influence aussi le non-alimentaire (durabilite, eco-conception)

### Processus de referencement

**Complexite supplementaire** : Carrefour fonctionne par formats (Hyper, Market, Express, City). Le referencement non-alimentaire concerne surtout les Hypers et potentiellement le e-commerce (carrefour.fr).

**Etapes :**
1. Identifier le CM PEM/Multimedia a la centrale (Massy)
2. Soumettre un dossier de referencement complet (fiche produit, tarifs, CGV, plan marketing)
3. Passage en comite de referencement (le CM presente a sa hierarchie)
4. Negociation des conditions : prix, promo, cooperation commerciale (RFA, NIP)
5. Integration SI et deploiement par format (hyper d'abord, puis e-commerce)
6. **Attention** : l'alliance Envergure peut complexifier le processus (double validation Carrefour + Systeme U pour certaines categories)

**Delai moyen : 4 a 9 mois**

### Comment aborder un acheteur Carrefour

1. **Comprendre que vous n'etes pas prioritaire** : Le non-alimentaire est strategiquement secondaire chez Carrefour. Soyez patient et persistant.

2. **Jouer le volume** : Carrefour a 240+ hypers en France. Meme avec un lineaire reduit, les volumes potentiels sont enormes. Mettez en avant le potentiel de volume.

3. **Prix imbattable ou innovation forte** : Carrefour ne referencera pas un "me-too" au meme prix que la concurrence. Il faut soit etre moins cher, soit proposer une innovation que Fnac/Boulanger n'ont pas.

4. **E-commerce comme porte d'entree** : Proposez un test sur carrefour.fr avant le deploiement en hyper. Moins risque pour l'enseigne, plus rapide a mettre en place.

5. **Integrer la RSE** : La strategie Carrefour pousse fortement la durabilite. Indice de reparabilite, packaging recyclable, certifications -- tout compte.

## E.LECLERC -- Organisation Achats

### Structure achats

E.Leclerc (CA ~56 Mds EUR, ~750 hypers + drives) est le 1er distributeur francais en CA. Son modele cooperatif unique rend le processus de referencement different de tous les autres.

**Le modele Leclerc : 3 niveaux**

| Niveau | Structure | Role | Infos |
|--------|-----------|------|-------|
| National | **GALEC** (Ivry-sur-Seine) | Centrale de referencement nationale. Negocie les conditions, reference les fournisseurs, mais n'impose PAS les achats aux magasins | 817+ collaborateurs |
| Regional | **SCA** (Scapnor, Scaouest, Lecasud...) | Centrales regionales. Peuvent referencement des fournisseurs locaux/regionaux en complement du Galec | 16 SCA en France |
| Local | **Adherent** (proprietaire du magasin) | Le chef d'entreprise independant decide LIBREMENT de son assortiment parmi les fournisseurs references | Chaque adherent est un decideur |

**Specificites Leclerc :**
- **Centrale de REFERENCEMENT** (pas d'achat) : le Galec negocie les conditions, mais chaque adherent decide d'acheter ou non
- Plateforme **SupplierXM** : canal unique pour le referencement fournisseur (declaration produits, fiches techniques, tarifs)
- Le non-alimentaire (multimedia, electromenager) est gere par des equipes dediees au Galec
- Michel-Edouard Leclerc a identifie le non-alimentaire comme relais de croissance ("c'est le secteur qui va tirer la consommation demain")
- La concurrence prix est absolue -- Leclerc se positionne comme le moins cher

### Processus de referencement

**Etape 1 -- Referencement national via le GALEC**
- S'inscrire sur la plateforme SupplierXM
- Completer les fiches produits (toutes les donnees sont obligatoires)
- Les tarifs sont publies dans un second temps, une fois les fiches matures
- Le Galec evalue : prix, marge, potentiel de rotation, concurrence existante

**Etape 2 -- Negociation des conditions commerciales**
- Convention unique annuelle avec le Galec
- Conditions : prix d'achat, RFA, NIP (Nouveaux Instruments Promotionnels), cooperation commerciale
- Cadre legal strict (loi EGAlim, transparence "ligne par ligne")

**Etape 3 -- Activation aupres des adherents**
- Une fois reference au Galec, vous devez ENCORE convaincre les adherents d'acheter
- Le KAM ou les forces de vente terrain doivent visiter les magasins, presenter les produits aux chefs de rayon
- Les SCA regionales peuvent faciliter cette activation

**Etape 4 -- Suivi et defense du lineaire**
- Les adherents peuvent dereferencent localement a tout moment
- Suivi terrain indispensable pour maintenir la presence

**Delai moyen : 4 a 12 mois** (le plus long, a cause du double processus national + local)

### Comment aborder le GALEC et les adherents Leclerc

1. **Respecter la plateforme SupplierXM** : Pas de raccourci. Le processus digital est obligatoire. Fiches produits completes, professionnelles, avec tous les visuels.

2. **Argumentaire prix imbattable** : Leclerc se bat sur le prix. Si votre prix public est plus eleve que chez Fnac ou Boulanger, il faudra expliquer pourquoi (exclusivite, pack specifique, etc.).

3. **Penser "adherent"** : Un referencement Galec sans activation terrain = 0 vente. Prevoyez une force de vente capable de visiter les magasins et convaincre les adherents locaux.

4. **Proposer des animations en magasin** : Les adherents Leclerc adorent les evenements qui generent du trafic. Journees de demonstration, promotions flash, corners dedies.

5. **Le drive comme levier** : E.Leclerc est leader du drive en France. Assurez une visibilite optimale sur le canal drive (fiches produits, mots-cles, visuels).

## AMAZON FR -- Organisation Achats

### Structure achats

Amazon France (CA estime ~10 Mds EUR) fonctionne radicalement differemment des enseignes physiques. Deux modeles coexistent :

**Vendor Central (1P -- First Party)** : Amazon achete vos produits et les revend. C'est le modele qui interesse Modelabs en priorite.

**Seller Central (3P -- Third Party)** : Vous vendez directement sur la marketplace Amazon. Modelabs peut aussi utiliser ce canal.

| Niveau | Role | Perimetre | Specificites |
|--------|------|-----------|-------------|
| Direction | Retail Category Leader | Strategie d'une grande categorie (ex: Electronics) | Decide des orientations strategiques |
| VM Senior | Senior Vendor Manager | Plusieurs sous-categories ou gros comptes | Negociateur principal pour les marques strategiques |
| VM | Vendor Manager | Portefeuille de marques/fournisseurs | Contact quotidien du KAM. Negocie prix, promos, terms |
| AVS | Amazon Vendor Services | Support premium (payant) | Account manager dedie pour les gros vendeurs |
| In-stock | In-Stock Manager | Gestion des stocks et commandes | Decide des quantites commandees (algorithmique) |

**Specificites Amazon :**
- **Tout est data-driven** : chaque decision est basee sur des metriques (ASIN performance, Buy Box, conversion rate, CRaP -- Can't Realize a Profit)
- **Invitation only** : pour Vendor Central, c'est Amazon qui vous invite. Mais vous pouvez proactivement contacter un Vendor Manager via les salons ou le reseau
- **Negociation annuelle** : Amazon negocie une fois par an les "terms" (conditions commerciales). Le reste de l'annee, le focus est sur l'execution
- **Automatisation** : les commandes sont largement automatisees (algorithmes). Le Vendor Manager pilote les conditions, pas les commandes individuelles
- **Amazon Retail Analytics** : donnees de vente detaillees disponibles pour les vendors (version Premium payante)

### Processus de referencement

**Option 1 -- Etre invite sur Vendor Central**
- Amazon identifie les marques a potentiel (souvent via les ventes Seller Central ou la notoriete marche)
- Un Vendor Manager vous contacte avec une proposition
- Negociation des "terms" : cost price, payment terms, freight, co-op (accruals)
- Setup des ASINs (fiches produits) sur Vendor Central
- Amazon passe les premieres commandes

**Option 2 -- Approche proactive**
- Demarrer sur Seller Central (FBA -- Fulfilled by Amazon) pour prouver le potentiel
- Contacter les Vendor Managers via CES, IFA, ou le reseau LinkedIn
- Presenter des volumes et une marque avec de la traction
- Amazon migre les marques a potentiel vers Vendor Central

### Comment aborder un Vendor Manager Amazon

1. **Parler en metriques Amazon** : ASIN, Buy Box %, conversion rate, CRaP score, glance views. Si vous ne parlez pas ce langage, vous perdez en credibilite immediatement.

2. **Contenu A+ obligatoire** : Amazon est un catalogue geant. Vos fiches produits doivent etre parfaites : titres optimises, bullet points clairs, images HD (minimum 6), contenu A+ (Enhanced Brand Content), video produit.

3. **Penser rentabilite Amazon** : Amazon surveille le CRaP (produits non rentables). Si votre produit est trop lourd, trop volumineux, ou a une marge trop faible apres frais logistiques -- Amazon le dereference. Calculez la rentabilite "Amazon-style" avant de proposer vos prix.

4. **Investir en Amazon Advertising** : Le Vendor Manager attend un investissement publicitaire (Sponsored Products, Sponsored Brands, DSP). Prevoyez un budget ads de 5-15% du CA Amazon.

5. **Etre reactif et structure** : Les VM gerent des dizaines de marques. Repondez vite, envoyez des documents clairs, respectez les deadlines. Un fournisseur lent est un fournisseur oublie.

## Guide Relationnel -- Bonnes Pratiques

### Les 10 regles d'or du KAM avec les acheteurs

**1. Preparez-vous comme un athlete**
Chaque rendez-vous se prepare comme une finale. Connaissez les chiffres de votre categorie (panels GfK/Circana), les performances de vos produits et de ceux des concurrents, les tendances du marche. L'acheteur respecte celui qui connait mieux son marche que lui.

**2. Ecoutez plus que vous ne parlez**
La regle du 70/30 : 70% d'ecoute, 30% de parole. Posez des questions ouvertes : "Quels sont vos enjeux sur cette categorie cette annee ?", "Qu'est-ce qui manque dans votre assortiment actuel ?". L'acheteur vous donnera les cles pour construire votre argumentaire.

**3. Ne bradez jamais sans contrepartie**
Chaque concession doit etre echangee. Si l'acheteur demande une baisse de prix, demandez un elargissement de gamme, un meilleur emplacement en rayon, ou un engagement de volume. La negociation est un echange, pas un abandon.

**4. Soyez fiable a 100%**
La fiabilite est la monnaie de la confiance. Livrez a l'heure, respectez vos engagements de prix, envoyez les documents demandes dans les delais. Un fournisseur fiable est un fournisseur qu'on garde, meme si son prix n'est pas le plus bas.

**5. Apportez de la valeur a chaque contact**
Ne contactez jamais un acheteur "juste pour prendre des nouvelles". Chaque interaction doit apporter quelque chose : une donnee marche, une innovation, une solution a un probleme. Devenez une ressource, pas une contrainte.

**6. Comprenez ses objectifs personnels**
L'acheteur a des objectifs individuels (marge, CA, reduction des references, innovation). Si vous comprenez SES objectifs, vous pouvez positionner votre offre comme une solution a SA reussite professionnelle.

**7. Respectez la hierarchie et les processus**
Ne court-circuitez jamais un acheteur pour aller voir son directeur. C'est l'erreur fatale. Si vous avez besoin d'un arbitrage hierarchique, faites-le avec l'accord de votre interlocuteur.

**8. Investissez dans le long terme**
Les meilleurs partenariats se construisent sur 3-5 ans. Acceptez de ne pas tout gagner la premiere annee. Un test reussi sur 20 magasins vaut mieux qu'un referencement national qui echoue.

**9. Soyez transparent sur les difficultes**
Un retard de livraison, un probleme qualite, une rupture de stock : prevenez l'acheteur AVANT qu'il ne le decouvre. La transparence dans la difficulte renforce la confiance plus que la perfection permanente.

**10. Connaissez vos concurrents mieux qu'eux-memes**
Sachez ce que proposent les marques concurrentes referencees chez l'enseigne : prix, conditions, parts de lineaire, promotions. Votre argumentaire doit se positionner PAR RAPPORT a la concurrence, pas dans l'absolu.

### Erreurs fatales a eviter

**1. Arriver sans connaitre les chiffres de l'enseigne**
Parler de "marche porteur" sans citer les panels, les parts de marche de l'enseigne sur la categorie, ou les performances sell-out de vos produits. L'acheteur considere que vous n'avez pas fait vos devoirs.

**2. Court-circuiter l'acheteur**
Aller voir le directeur commercial sans passer par le CM ou l'acheteur. Vous obtiendrez peut-etre un rendez-vous... et un ennemi definitif en la personne de l'acheteur contourne.

**3. Promettre et ne pas livrer**
Annoncer un prix, un delai, un budget marketing que vous ne pouvez pas tenir. En grande distribution, la memoire est longue. Un engagement non tenu vous suivra pendant des annees.

**4. Negliger le terrain apres le referencement**
Obtenir le referencement et ne jamais visiter les magasins. Les chefs de rayon ne mettront pas en avant un produit dont ils n'ont jamais vu le representant. Le referencement sans activation terrain = echec.

**5. Reagir emotionnellement aux techniques de pression**
L'acheteur dit "c'est trop cher, votre concurrent fait -15%". Vous paniquez et baissez votre prix. Grave erreur. Prenez du recul, demandez des preuves, et proposez un compromis structure.

**6. Ne penser qu'au prix**
Reduire la negociation au prix d'achat, c'est jouer le jeu de l'acheteur. Elargissez la discussion : services, formation, marketing, logistique, exclusivites. Plus la discussion est large, plus vous avez de leviers.

**7. Ignorer la saisonnalite**
Proposer des trottinettes electriques en novembre ou des chauffages en avril. Chaque categorie a son calendrier. Respectez-le ou vous perdez votre credibilite.

### Comment construire la confiance

**Phase 1 -- Premier rendez-vous (mois 0-3)**
- Objectif : credibilite. Montrez que vous connaissez le marche, l'enseigne, et la categorie
- Apportez des donnees, des echantillons, un plan d'affaires structure
- Ne demandez pas trop : un test sur quelques magasins ou en e-commerce
- Suivi : envoyez un compte-rendu structure dans les 24h

**Phase 2 -- Test et preuves (mois 3-12)**
- Objectif : fiabilite. Livrez impeccablement, respectez chaque engagement
- Fournissez des reportings reguliers (ventes, stocks, satisfaction client)
- Reagissez vite aux problemes (SAV, logistique)
- Proposez des ajustements bases sur les premiers resultats

**Phase 3 -- Developpement (annee 2)**
- Objectif : partenariat. Proposez un plan de developpement sur 2-3 ans
- Elargissez la gamme progressivement
- Co-developpez des operations marketing (exclusivites, lancements evenementiels)
- Partagez votre roadmap produit en avant-premiere

**Phase 4 -- Partenariat strategique (annee 3+)**
- Objectif : incontournabilite. Devenez le fournisseur de reference de votre categorie
- Participez aux reflexions strategiques de l'enseigne (category reviews)
- Proposez des innovations en co-developpement
- Construisez des relations a tous les niveaux (CM, acheteur, direction, terrain)
