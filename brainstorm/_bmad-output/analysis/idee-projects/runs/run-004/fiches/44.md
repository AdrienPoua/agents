## 1. CONTEXTE METIER

### C'est quoi ce metier/secteur ?

Les **professionnels de l'immobilier** en France (agents immobiliers, mandataires, administrateurs de biens, syndics de copropriete, diagnostiqueurs) exercent une activite **fortement reglementee** depuis la loi Hoguet du 2 janvier 1970. Chaque professionnel doit detenir une **carte professionnelle CPI** delivree par la CCI, renouvelable tous les 3 ans.

Le secteur represente en 2025 :
- **62 750 detenteurs directs** de cartes T (transaction), G (gestion), S (syndic)
- **~80 424 cartes** delivrees pour agents commerciaux/mandataires
- **~11 581 diagnostiqueurs** certifies (source : annuaire DIN, mars 2025)
- **Total estime : ~203 920 professionnels** soumis a des obligations de conformite

### Pourquoi il y a un probleme maintenant ?

1. **Multiplication des obligations** : La loi ALUR (2014) a ajoute la formation continue obligatoire (42h/3 ans), la loi ELAN (2018) a renforce le CNTGI et les controles
2. **Controles CCI systematises** : En 2026, les CCI verifient systematiquement les justificatifs de formation lors des renouvellements de cartes
3. **Baisse du marche** : -15 847 cartes professionnelles delivrees en 2025 vs 2024, mettant les professionnels sous pression financiere (moins de marge pour l'erreur)
4. **Complexite croissante** : Entre carte pro (3 ans), formation ALUR (14h/an ou 42h/3 ans dont 2h deontologie + 2h non-discrimination), RC Pro (annuel), garantie financiere (annuel), registre des mandats (permanent), les echeances s'empilent

### Contexte reglementaire

| Texte | Contenu cle | Impact conformite |
|-------|-------------|-------------------|
| **Loi Hoguet (1970)** | Carte professionnelle obligatoire, garantie financiere, registre des mandats | Cadre fondateur, toujours en vigueur |
| **Decret du 20/07/1972** | Art. 3-1 : formation continue 42h/3 ans ; Art. 72 : registre mandats | Echeances recurrentes |
| **Loi ALUR (2014)** | Formation continue, CNTGI, code deontologie | Ajout d'obligations periodiques |
| **Decret 28/08/2015** | Code de deontologie des professionnels | 2h formation deontologie obligatoire |
| **Loi ELAN (2018)** | Refonte CNTGI, renforcement controles | Commission de controle active |
| **Decret 14/10/2020** | Formation non-discrimination | 2h supplementaires obligatoires |

> Sources : [Loi Hoguet obligations 2026 - Maformationimmo](https://www.maformationimmo.fr/loi-hoguet-obligations-2026-et-erreurs-qui-exposent-les-agences-a-des-sanctions/), [Loi Hoguet - Coursange Avocats](https://www.coursange-avocats.com/actualites/loi-hoguet-tout-savoir/), [Loi ALUR 2025 - Proprietes Privees](https://www.proprietes-privees.org/loi-alur-2025-formation-initiale/)

## 2. LA DOULEUR

### Qui souffre exactement ?

| Segment | Nombre estime | Profil type |
|---------|---------------|-------------|
| Agents immo independants (carte T) | ~30 000 | Gerant d'agence seul ou petite equipe, pas de service juridique |
| Mandataires (agents commerciaux) | ~80 000 | Independants sous reseau (IAD, Capifrance...), zero support admin |
| Administrateurs de biens (carte G) | ~8 000 | Gestion locative, lourdes obligations garantie financiere |
| Syndics (carte S) | ~5 000 | Copropriete, garantie financiere 110 000 EUR min |
| Diagnostiqueurs immobiliers | ~11 500 | Certification 7 ans, assurance, formation continue |
| **TOTAL cible directe** | **~135 000** | Professionnels avec echeances reglementaires multiples |

### De quoi souffrent-ils ?

Le probleme est **silencieux mais devastateur** :

1. **Echeances multiples desynchronisees** : La carte pro expire tous les 3 ans (date variable par professionnel), la RC Pro se renouvelle annuellement (date differente), la garantie financiere a sa propre echeance, la formation ALUR se cumule sur 3 ans glissants
2. **Pas de systeme d'alerte centralise** : La CCI n'envoie pas de rappel. Les assureurs envoient un courrier 1 mois avant, souvent perdu
3. **Demande de renouvellement 2 mois avant expiration** : Oublier = se retrouver sans carte = exercice illegal
4. **Gestion documentaire eclatee** : Attestations RC Pro dans les mails, carte pro en papier, justificatifs formation sur differentes plateformes

### Combien ca coute ?

| Risque | Impact |
|--------|--------|
| Exercice sans carte professionnelle | **6 mois de prison + 7 500 EUR d'amende** (Art. 14 loi Hoguet) |
| Radiation du fichier national | **Arret total d'activite** jusqu'a regularisation (1 an max pour re-deposer) |
| Absence de RC Pro | **Exercice illegal** + responsabilite personnelle illimitee en cas de litige |
| Absence garantie financiere | **Impossible de recevoir des fonds** (acomptes, depots de garantie, loyers) |
| Formation non validee | **Refus de renouvellement** de la carte par la CCI |
| Temps de gestion manuelle | **8-15h/an** de recherche de documents, appels assureurs, verification dates |

### Cas concrets

- Un agent qui oublie de demander son renouvellement 2 mois avant expiration se retrouve **radie du fichier national des professionnels de l'immobilier** et doit cesser toute activite
- La CCI Savoie a publie un [rappel de procedure de radiation](https://www.savoie.cci.fr/actualite/rappel-de-la-procedure-de-radiation-pour-la-cessation-de-garantie-etou-dassurance-des) pour cessation de garantie et/ou d'assurance, confirmant que les radiations effectives existent
- Le CNTGI, via sa commission de controle, transmet les cas d'infraction a la [DGCCRF](https://www.anil.org/aj-cntgi-organisation-fonctionnement-procedure-disciplinaire/) pour poursuites

1. **Inscription** (2 min) : Email + type de carte (T/G/S) + date d'expiration carte + date RC Pro
2. **Configuration** (5 min) : Ajout garantie financiere, heures formation ALUR, upload documents existants
3. **Dashboard actif** : Vue synthetique de toutes les echeances, code couleur, prochaine action requise
4. **Alerte recue** : Email/notif "Votre RC Pro expire dans 60 jours - Voici la checklist de renouvellement"
5. **Action guidee** : Checklist pas-a-pas du renouvellement avec liens directs vers CCI, assureurs, organismes formation

**Verdict** : Le pattern "calendrier de conformite SaaS" est **valide internationalement** (eCesis, compliance calendars) mais **n'existe pas en version specialisee pour les professionnels de l'immobilier francais**. C'est un ocean blue confirme.

### Mitigations

1. Contenu SEO + checklist gratuite = eduquer + capter avant la douleur
2. Valeur continue : veille reglementaire + coffre-fort + multi-echeances annuelles (pas juste la carte)
3. First-mover advantage + specialisation = les CRM ne prioriseront pas ce module
4. Meme en baisse, 135 000+ professionnels = marche largement suffisant pour objectif 3 000 EUR/mois
