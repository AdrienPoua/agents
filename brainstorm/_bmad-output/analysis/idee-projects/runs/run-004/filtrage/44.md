## 2. PREUVE DE DOULEUR - Score : 3.5/5 (Moderee, structurelle)

### Preuves trouvees

**Sanctions reelles et documentees :**
- Exercice sans carte T valide = **6 ans de prison + 7 500 EUR amende** (article 14 loi Hoguet)
- Perte du droit aux commissions sur transactions realisees sans carte valide
- CCI radie automatiquement du registre national a expiration
- Sources : [Avocats Picovschi](https://immobilier.avocats-picovschi.com/agent-immobilier-et-absence-de-carte-professionnelle-un-risque-garanti.html), [MaFormationImmo](https://www.maformationimmo.fr/les-consequences-du-non-renouvellement-de-la-carte-dagent-immobilier-un-risque-a-ne-pas-prendre/)

**Complexite des echeances :**
- Carte T/G/S : renouvellement tous les 3 ans, demande 2 mois avant expiration
- Formation ALUR : 42h sur 3 ans dont 2h deontologie + 2h non-discrimination obligatoires
- RC Pro : renouvellement annuel, pas de carte sans attestation
- Garantie financiere : renouvellement annuel, amende 7 500 EUR si absente
- Source : [FNAIM](https://www.fnaim.fr/espace-adhesion/3556-etape-3-souscrire-garantie-financiere-et-assurance-rcp-immobilier.htm)

**Probleme** : La douleur est REELLE mais DIFFUSE. Les agents n'en parlent pas sur les forums car c'est un risque silencieux - ils decouvrent le probleme quand c'est trop tard. Peu de plaintes publiques visibles mais les consequences sont catastrophiques.

**Baisse 2025** : -15 847 cartes professionnelles delivrees vs 2024 = marche sous pression, plus d'oublis potentiels.
- Source : [Immo Matin](https://www.immomatin.com/franchise/reseaux-franchise/immobilier-15-847-cartes-professionnelles-delivrees-par-les-cci-en-2025-par-rapport-a-2024.html)

**Score : 3.5/5** - Douleur structurelle confirmee par les sanctions, mais peu de temoignages publics (risque silencieux).
