## 4. DECLENCHEUR VERIFIABLE (Score: 2/5 - Obligation sans sanctions appliquees aux petits)

### Amendes THEORIQUES (ce que les articles disent)
- Mentions legales absentes : 75 000EUR (EI) / 375 000EUR (societes) + 1 an d'emprisonnement
  - Source : https://entreprendre.service-public.gouv.fr/vosdroits/F31228
- CGV absentes : 15 000EUR (e-commerce)
  - Source : https://www.keobiz.fr/le-mag/condition-generale-vente-e-commerce/
- Non-conformite info precontractuelle : 3 000EUR (personne physique)
  - Source : https://www.gouache.fr/ressources/articles-avocat-les-controles-dgccrf-sur-les-sites-e-commerce/

### Sanctions REELLEMENT APPLIQUEES (jurisprudence)
- **TGI Paris 2014** : 6 000EUR x2 pour 2 editeurs d'un site de notation d'entreprises (notetonentreprise.com). Mais c'etait parce qu'un tiers ne pouvait pas identifier l'editeur pour exercer son droit de reponse. Pas un controle DGCCRF proactif.
  - Source : https://www.haas-avocats.com/plateformes/ecommerce/6-000-euros-condamnation-absence-mentions-legales-site-web/
- **Tribunal correctionnel 2017** : Condamnation de l'editeur du site Egalite et Conciliation. Contexte politique, pas commercial.
  - Source : https://www.lagbd.org/Sites_internet_l_absence_de_mentions_legales_est_un_delit_penal
- **Tourmag.com 2024** : Condamnation pour non-respect des mentions legales. Site professionnel de presse touristique, pas un AE.
  - Source : https://www.lexplicite.fr/defaut-mentions-legales-conduire-condamnation-penale-lediteur-du-site/
- **Praticien shiatsu** : Controle DGCCRF a distance, injonction de modification sous 8 jours, amende administrative de 45 000EUR. MAIS : c'est une exception tres inhabituelle, et le site upsme.fr confirme que "les sanctions sont rarissimes".
  - Source : https://www.twothomas.com/post/contr%C3%B4le-dgccrf-comment-y-faire-face-et-pourquoi-faire-appel-%C3%A0-un-avocat

### Controles DGCCRF en 2024 : chiffres globaux
- 64 979 etablissements et sites internet controles
- 2 356 amendes administratives (92.9M EUR total)
- 21 769 avertissements, 8 625 injonctions
- MAIS : cible principale = e-commerce, marketplaces, influenceurs (260 influenceurs controles). **Pas les sites vitrine d'AE.**
  - Source : https://www.economie.gouv.fr/dgccrf/actualites-dgccrf/la-dgccrf-en-action-bilan-de-lactivite-2024-et-perspectives-2025

### Analyse critique du declencheur
1. **Les amendes de 75K sont THEORIQUES** : en jurisprudence, les condamnations reelles sont de 3 000 a 6 000EUR. Le montant de 75K est le maximum legal, jamais applique a un petit.
2. **La DGCCRF ne cible PAS les AE pour les mentions legales** : elle cible les e-commerce, les marketplaces, les faux avis, les influenceurs. Un site vitrine d'AE coach ou artisan est invisible pour la DGCCRF.
3. **Pas de deadline imminente** : contrairement a ConformLoc (deadline 20 mai 2026), il n'y a aucune echeance qui cree de l'urgence.
4. **Le cas du praticien shiatsu (45K EUR) est une exception** : meme les sources pro-conformite admettent que "les sanctions sont rarissimes".

**Score : 2/5** - Sanctions theoriques impressionnantes mais quasi-jamais appliquees aux petits. Zero deadline. Zero FOMO.
