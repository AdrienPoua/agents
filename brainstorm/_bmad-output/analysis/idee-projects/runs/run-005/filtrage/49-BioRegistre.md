# Filtrage Web - Idee #49 - BioRegistre

**Score initial:** 17.5/20
**Cible:** Restaurateurs et commerces alimentaires (200K+ cibles)
**Proposition:** Registre dechets non dangereux + tracabilite biodechets (loi AGEC 2024)

---

## 1. CONCURRENCE (Score: 2/5 - Competition moderee)

### Constats
- **Trackdechets (gouvernement, GRATUIT)** : Outil gouvernemental qui depuis mai 2025 a fusionne avec le RNDTS pour creer un registre national unique couvrant TOUS les types de dechets (dangereux ET non dangereux). Depuis janvier 2026, la declaration obligatoire sur Trackdechets est en vigueur pour les dechets non dangereux. L'outil est gratuit et permet deja le suivi des dechets non dangereux avec declaration automatique au registre.
- **Previsoft** : Module gestion des dechets (registre, suivi couts, fiches de suivi)
- **Neurowaste** : Logiciel SaaS gestion des dechets
- **Endrotek / Waste Monitor** : Solution SaaS + materielle, generation registres reglementaires
- **Octopus HACCP** : App tracabilite HACCP pour restaurants, inclut gestion biodechets/gaspillage
- **Upcycle** : Composteurs + logiciel de gestion, tracabilite et reporting automatique
- **Axibio** : Logiciel tracabilite biodechets avec generation automatique du registre dechets non dangereux

### Verdict concurrence
Le marche est deja couvert sur DEUX fronts :
1. **Cote registre reglementaire** : Trackdechets (GRATUIT, gouvernemental) couvre exactement le besoin
2. **Cote operationnel biodechets** : Moulinot, Upcycle, Axibio fournissent tracabilite + collecte

---

## 2. PREUVE DE DOULEUR (Score: 2/5 - Quelques temoignages)

### Constats
- Peu de plaintes trouvees en ligne specifiquement sur le registre dechets non dangereux
- La douleur principale des restaurateurs est le TRI des biodechets (operationnel), pas le registre (administratif)
- Les collecteurs (Moulinot, etc.) fournissent deja les bons de collecte avec tracabilite
- Les restaurateurs sont plus concernes par HACCP, hygiene, controles sanitaires que par le registre dechets
- 1 an apres la loi AGEC (janvier 2024), le sujet biodechets est traite mais via la collecte, pas via un logiciel registre

### Verdict douleur
La douleur est diffuse. Les restaurateurs galere avec le TRI physique des biodechets, pas avec la tenue du registre. Le registre est un sous-produit de la collecte, pas un besoin standalone.

## 4. DECLENCHEUR VERIFIABLE (Score: 3/5 - Obligation reelle, sanctions theoriques)

### Constats
- **Loi AGEC** : Tri a la source des biodechets obligatoire depuis 01/01/2024 pour tous les professionnels
- **Registre dechets non dangereux** : Obligatoire depuis 2011 (article R541-43 Code environnement)
- **Sanctions** : Jusqu'a 75 000 EUR d'amende et 2 ans de prison (theoriquement)
- **Trackdechets** : Declaration au registre national obligatoire avec delai de 7 jours
- **Controle DDPP** : Reel mais semble focalise sur hygiene alimentaire plus que sur registre dechets
- Pas de deadline imminente nouvelle - l'obligation est deja en vigueur depuis 2024

### Verdict declencheur
L'obligation existe et les sanctions sont sur le papier, mais pas de deadline imminente. Le declencheur est passe (janvier 2024).

## DECISION : ELIMINER

### Raison principale
**RED FLAG : Outil gouvernemental GRATUIT (Trackdechets) couvre le besoin exact.** Depuis mai 2025, Trackdechets a fusionne avec le RNDTS et couvre desormais les dechets non dangereux. Depuis janvier 2026, la declaration est obligatoire sur cette plateforme gratuite. C'est exactement le pattern "government free tool kills the idea" deja observe avec OiRA/OPPBTP sur DUERP Express et Passeport Prevention sur FormaPilote.

### Raisons secondaires
- La douleur est sur le tri physique, pas sur le registre administratif
- Les collecteurs (Moulinot, Upcycle, Axibio) incluent deja tracabilite + reporting
- Pas de deadline imminente = pas d'urgence d'achat
- Octopus HACCP couvre deja les restaurateurs sur la compliance alimentaire

### Pattern identifie
"Government free tool" (Trackdechets) + "Bundled with existing service" (collecteurs incluent tracabilite) = double menace mortelle.
