# Filtrage Web - Idee #56 - ConformERP5

**Cible**: ERP categorie 5 (600-800K petits commerces, restaurants, salons, cabinets)
**Proposition**: SaaS simple de conformite securite pour petits ERP (registre securite, verifications periodiques, plan intervention)

---

## 1. CONCURRENCE (Score: 2/5 - Concurrents existants et bien positionnes)

### Resultats de recherche
- **Deemply** (ex-registresecurite.com) :
  - Leader du registre de securite dematerialise
  - S'adapte a TOUT type de site : ERT, ERP, habitation, IGH, IMH
  - **Offre GRATUITE** lancee en octobre 2025 : registre de securite gratuit, sans carte bleue
  - Modules : registre securite, DUERP, carnet sanitaire, registre accessibilite, permis feu, plan prevention, gestion batiment
  - Couvre EXACTEMENT le besoin de ConformERP5
- **BatiFire** :
  - Solution digitale pour piloter la conformite des batiments
  - Referencement gratuit des batiments dans le module Secours
- **Registresecurite.com** : Desormais fusionne avec Deemply
- **MON-ERP.fr** : Guide de conformite ERP en ligne (informatif, pas SaaS)
- **Anco, Preveris** : Prestataires de controle technique, pas SaaS

### Analyse
**Deemply est un RED FLAG MAJEUR** :
- C'est exactement le meme positionnement que ConformERP5
- Ils ont lance une offre GRATUITE en octobre 2025
- Ils couvrent tous les types d'ERP dont la categorie 5
- Ils ont de l'historique (ex-registresecurite.com)
- Pattern identifie dans les runs precedents : "Leader who offers free tier" = tue le nouvel entrant

**Score: 2/5** - Deemply couvre le besoin exact avec une offre gratuite.

## 2. PREUVE DE DOULEUR (Score: 3/5 - Douleur reelle mais passive)

### Resultats de recherche
- Les articles sur les obligations ERP cat5 sont nombreux et pedagogiques.
- L'arrete du 1er decembre 2025 cree de nouvelles obligations = potentiel de confusion.
- **MAIS** : Pas de plaintes massives en ligne de commercants sur la gestion de leur registre de securite.
- Les ERP cat5 sans locaux a sommeil ne font PAS l'objet de visites periodiques de la commission de securite.
- Le controle se fait uniquement : a l'ouverture/reouverture ou de maniere inopinee (denonciation, initiative du maire).
- **Signal faible** : Les commercants cat5 ne savent souvent meme pas qu'ils sont un ERP.

### Analyse
La douleur existe "sur le papier" (complexite reglementaire, nouvelles obligations) mais dans la realite :
- Pas de controles periodiques pour les ERP cat5 sans locaux a sommeil
- Les controles sont rares (inopines uniquement)
- Les commercants ignorent souvent leurs obligations = pas de douleur ressentie

**Score: 3/5** - Douleur theorique forte mais douleur ressentie faible car peu de controles.

## 4. DECLENCHEUR VERIFIABLE (Score: 3.5/5 - Arrete confirme mais application diluee)

### Resultats de recherche
- **Arrete du 1er decembre 2025** : CONFIRME sur Legifrance (JORFTEXT000053020948)
  - Publie au Journal Officiel le 14 decembre 2025
  - Entre en vigueur le 1er janvier 2026
  - Verification periodique des installations de gaz : au moins tous les 3 ans (NOUVEAU pour cat5)
  - Installations neuves/modifiees : verification obligatoire a partir du 1er juillet 2026
- **Amendes** :
  - 45 000 EUR pour l'exploitant = CONFIRME mais c'est le maximum legal pour tous les ERP, pas specifique cat5
  - 750 EUR pour mauvaise tenue du registre de securite
  - Fermeture administrative possible
- **MAIS** : Pas de controles periodiques de la commission de securite pour les ERP cat5 sans locaux a sommeil
  - Controles uniquement : a l'ouverture ou inopines
  - En pratique, les ERP cat5 sont tres rarement controles

### Analyse
Le decret est reel et les sanctions existent "sur le papier". MAIS :
- 45 000 EUR est le maximum legal theorique, rarement applique aux petits commerces
- 750 EUR pour mauvaise tenue du registre est plus realiste
- Pas de commission de securite periodique = pas de "moment de verite" regulier
- Pattern des runs precedents : "sanctions sur le papier vs sanctions appliquees" = distinction critique
- Ce pattern rappelle les ERP cat5 AccessiBoard (Run 3) : sanctions existantes mais 0 appliquees

**Score: 3.5/5** - Arrete reel, sanctions theoriquement fortes, mais application quasi inexistante pour les cat5.

## VERDICT: ELIMINER

**Raison principale** : Deemply (ex-registresecurite.com) couvre EXACTEMENT le meme besoin avec une offre GRATUITE lancee en octobre 2025. C'est le pattern "leader with free tier" identifie dans les runs precedents. De plus, les ERP cat5 ne font pas l'objet de controles periodiques, ce qui reduit l'urgence. Les sanctions de 45 000 EUR sont theoriques et quasiment jamais appliquees aux petits commerces. Combinaison fatale : concurrent gratuit + pas de controles reels + overlap avec ConformCHR.
