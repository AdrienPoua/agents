# Filtrage Web - Idee #54 - CertifVert

**Cible**: Entreprises du paysage utilisant des produits phytopharmaceutiques (15-20K cibles)
**Proposition**: SaaS de gestion conformite phyto pour paysagistes (certiphyto, agrement DRAAF, registre phyto numerique)

---

## 1. CONCURRENCE (Score: 2/5 - Competition moderee)

### Resultats de recherche
- **Smag (Farmer, LEA)** : Logiciel agricole leader. Gere le registre phyto, l'agrement phyto, les certifications. INCLUT deja les prestataires/paysagistes dans leur cible ("ETA, paysagistes, pepinieristes").
- **Easyphyt** (par Uneal) : Solution d'enregistrement des donnees phyto avec controle reglementaire. Conforme au format du registre numerique.
- **MesParcelles** : Logiciel de gestion parcellaire avec registre phyto integre.
- **Geofolia, Farmi** : Autres logiciels agricoles conformes a l'obligation registre phyto numerique.
- **Organilog, Robbin, JardiContacts, PaysagisteModerne** : Logiciels de gestion pour paysagistes (devis, factures, planning chantier) mais SANS module registre phyto.

### Analyse
Le marche est coupe en deux :
- **Solutions agricoles (Smag, Easyphyt, MesParcelles)** : Deja conformes au registre phyto numerique 2027, et elles INCLUENT explicitement les paysagistes/ETA dans leur cible
- **Solutions paysagistes (Organilog, Robbin)** : Gestion de chantier mais pas de module phyto

Le probleme : les solutions agricoles COUVRENT deja le besoin registre phyto pour les paysagistes. Le "gap" n'est pas aussi grand qu'annonce.

**Score: 2/5** - Smag et Easyphyt couvrent deja le registre phyto pour les paysagistes.

## 2. PREUVE DE DOULEUR (Score: 2/5 - Peu de temoignages)

### Resultats de recherche
- **Pas de plaintes specifiques** de paysagistes sur la gestion phyto trouvees sur les forums.
- Les articles sont pedagogiques/informatifs (Chambres d'Agriculture) mais pas de temoignages de douleur.
- Le sujet "registre phyto numerique" est aborde presque exclusivement sous l'angle agricole.
- Probleme mentionne : "le tarif des outils informatiques et le manque d'aisance PC" pour ceux encore sur papier - mais c'est un probleme agriculteurs, pas specifique paysagistes.

### Analyse
Les paysagistes ne semblent pas se plaindre massivement de la gestion phyto en ligne. Le bruit est cote agricole. Les paysagistes representent une niche dans la niche de la gestion phyto.

**Score: 2/5** - Silence quasi total des paysagistes sur ce sujet.

---

## 3. TAILLE DE MARCHE (Score: 2.5/5 - Niche dans la niche)

### Resultats de recherche
- **33 550 entreprises du paysage en France en 2024** (source UNEP/Valhor)
- **61.5% sans salarie** = micro-entreprises
- MAIS : tous les paysagistes n'utilisent PAS de produits phytopharmaceutiques
  - Depuis 2019 : interdiction des phytos chimiques pour les particuliers/amateurs
  - Les paysagistes professionnels peuvent encore utiliser des produits phyto MAIS la tendance est au zero-phyto
  - Seuls les paysagistes avec certiphyto + agrement DRAAF utilisent des phytos
- Estimation cible realiste : **5-15K** (ceux qui ont reellement un certiphyto et utilisent des phytos), pas 15-20K

### Analyse
Le chiffre de 15-20K est optimiste. Avec la tendance zero-phyto et l'interdiction croissante, le marche cible DIMINUE chaque annee. C'est un marche en contraction.

**Score: 2.5/5** - Probablement 5-15K et en diminution.

---

## 4. DECLENCHEUR VERIFIABLE (Score: 3.5/5 - Deadline confirmee mais sanctions faibles)

### Resultats de recherche
- **Registre phyto numerique** : Obligation CONFIRMEE au 1er janvier 2027 (reportee de 2026)
  - Source : Journal Officiel, Legifrance, multiples articles presse agricole
  - Reglement europeen, transpose en droit francais
- **MAIS** : Phase transitoire 2027-2029 : les professionnels peuvent continuer a utiliser leurs outils actuels (meme papier!) a condition de convertir en format electronique avant le 31 janvier de l'annee suivante
- **Sanctions** : Amende administrative jusqu'a **1 500 EUR** + perte d'aides PAC + blocage certifications
  - Les aides PAC ne concernent PAS les paysagistes (pas des agriculteurs)
  - Donc pour un paysagiste : seulement l'amende de 1 500 EUR
- **Formats acceptes** : Excel (.xlsx), CSV, XML, exports logiciels homologues = Excel suffit!

### Analyse
La deadline est reelle mais pas urgente :
- Phase transitoire jusqu'a 2029 (papier encore tolere avec conversion)
- Un simple fichier Excel est conforme
- Sanctions faibles pour les paysagistes (pas d'aides PAC a perdre)
- La deadline pousse vers Smag/Easyphyt, pas vers un nouvel entrant

**Score: 3.5/5** - Deadline reelle mais diluee par la transition et la facilite de conformite via Excel.

## RED FLAGS
1. **Smag et Easyphyt couvrent deja les paysagistes** : Ce ne sont pas "juste pour l'agriculture" - ils ciblent explicitement les ETA et paysagistes
2. **Un fichier Excel suffit** : Le registre phyto numerique accepte les formats Excel/CSV, pas besoin de SaaS
3. **Marche en contraction** : Tendance zero-phyto, interdictions croissantes = moins de cibles chaque annee
4. **Phase transitoire 2027-2029** : Papier tolere avec conversion annuelle, reduit l'urgence
5. **Sanctions faibles pour paysagistes** : 1 500 EUR max, pas de perte d'aides PAC

---

## VERDICT: ELIMINER

**Raison principale** : Concurrence existante (Smag, Easyphyt couvrent deja les paysagistes) + un fichier Excel suffit pour etre conforme + marche en contraction (tendance zero-phyto) + phase transitoire jusqu'a 2029 qui reduit l'urgence. Le "gap" entre solutions agricoles et besoin paysagiste est beaucoup plus mince que suppose.
