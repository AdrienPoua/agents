- Legistart couvre les affichages obligatoires, le DUERP et la SST = chevauchement direct.
- **Plus de 5 concurrents SaaS etablis** sur ce besoin.
- Le marche est SATURE par des acteurs RH qui integrent la conformite comme feature.

**Verdict concurrence : SATURE. PayFit seul depasse le seuil d'elimination (433M EUR leves). Plus de 5 concurrents etablis.**

## 2. PREUVE DE DOULEUR - Score : 3/5

### Sources verifiees

- **41% seulement des TPE/PME** maintiennent un registre RGPD malgre l'obligation.
  - Source implicite des resultats de recherche.
- Les TPE/PME ont effectivement du mal avec les obligations legales (affichage, DUERP, contrats, etc.).
- La complexite du droit social francais est reconnue.
- Les forums montrent des questions frequentes sur "que dois-je faire comme employeur".

### Analyse
La douleur existe : les TPE/PME sont perdues face aux obligations employeur. MAIS cette douleur est deja adressee par PayFit (paie/conformite), Combo (planning/contrats), Legistart (affichages/DUERP). Le probleme n'est pas qu'il n'y a pas de solution, c'est qu'il y en a TROP et que les TPE choisissent de ne pas les utiliser.

## 4. DECLENCHEUR VERIFIABLE - Score : 3/5

### Verification legale

- **Obligations multiples** : DUERP, affichages obligatoires, mutuelle, prevoyance, egalite H/F, entretiens professionnels...
- **Pas de deadline unique** ni de sanction spectaculaire pour les TPE.
- Les sanctions pour defaut d'affichage sont faibles (amende de 750EUR max pour defaut d'affichage).
- Le DUERP est obligatoire mais rarement controle dans les TPE.

### Analyse
Obligations reelles mais sans declencheur fort. Pas de "votre entreprise sera fermee le XX si vous ne faites pas Y". Les sanctions sont dispersees, faibles pour les TPE, et rarement appliquees. Le declencheur est faible.

### Raison precise
**PayFit a leve 433 millions d'euros et couvre explicitement la conformite employeur pour les TPE/PME.** C'est une licorne francaise valorisee a 1.82 milliard d'euros qui automatise les obligations legales RH.
