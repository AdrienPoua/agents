# Idee #35 - FleetSimple

**Angle** : 11 - Fragmentation d'outils
**Date** : 2026-02-10

---

## Fiche projet

| Champ | Detail |
|---|---|
| **Nom du projet** | FleetSimple |
| **Cible** | Patrons de TPE artisanales et PME de services (plombiers, electriciens, nettoyage, livraison, BTP) qui gerent 5 a 30 vehicules utilitaires |
| **Douleur** | Le patron gere sa flotte sur un fichier Excel bricolé : il oublie les dates de controle technique (135 EUR d'amende + immobilisation), rate les echeances d'assurance, ne sait pas quel vehicule est dispo, perd du temps a croiser les infos entre 4 sources (Excel km, emails assureur, SMS garagiste, carnet papier) |
| **Declencheur d'achat** | Un vehicule immobilise pour CT oublie (= journee perdue + amende), ou un sinistre non-assure parce que l'echeance est passee, ou un nouveau vehicule qui rentre dans la flotte et le fichier Excel devient ingerace |
| **Concurrents connus** | **FleetNote** (francais, cible TPE/PME, interface simple, transition depuis Excel -- concurrent le plus direct), **Winflotte** (plutot grands comptes, 100+ vehicules, complexe), **Cegid Notilus** (ERP complet, trop lourd pour une TPE artisanale), **Seven Fleet** (web, correct mais oriente suivi GPS/telematique), **Ocean** (telematique, boitiers GPS, hors budget TPE) |
| **Differentiation** | **Prix agressif** : 1-2 EUR/vehicule/mois (vs 5-15 EUR chez FleetNote, 20-50 EUR chez Winflotte). **Zero telematique** : pas de boitier GPS, pas de geolocalisation -- juste du suivi administratif pur (CT, assurance, entretien, km, affectation). **Onboarding en 5 min** : import Excel du parc existant, zero formation. **Alertes SMS/email** automatiques pour le patron ET le conducteur. **Cible ultra-precise** : le patron artisan qui a entre 5 et 30 vehicules, pas de fonctionnalites inutiles pour les flottes de 200+. |

---

## Score rapide /20

| Critere | Note | Justification |
|---|---|---|
| Intensite douleur | 3.5/5 | La douleur est reelle (amendes CT 135 EUR, sinistres non-couverts, temps perdu) mais pas vitale au quotidien. Le patron s'en sort avec Excel tant qu'il n'y a pas d'incident. Le declencheur est ponctuel, pas permanent. |
| Facilite acquisition | 4/5 | Cible tres accessible : annuaires metiers (FFB, CAPEB, FNTP), groupes Facebook artisans, salons pro, SEO sur "gestion flotte vehicules TPE". Possibilite de scraper les entreprises avec 5+ vehicules via les registres professionnels. Content marketing : "Modele Excel gestion flotte gratuit" comme lead magnet. |
| Simplicite vente | 4/5 | Argument visuel immediat : "Votre Excel de flotte avec des alertes automatiques, pour 1 EUR/vehicule/mois". Le patron comprend en 5 secondes. Le ROI est evident : 1 CT oublie = 135 EUR d'amende + 1 journee de vehicule immobilise > 1 an d'abonnement. |
| Faisabilite solo | 4.5/5 | MVP simple : CRUD vehicules + alertes date + dashboard. Stack Next.js + Supabase + Vercel. Pas d'integration complexe requise (pas de GPS, pas de telematique). Import CSV pour onboarding. MVP realisable en 2-3 semaines. |

**TOTAL : 16/20**

## Estimation marche cible France

- ~350 000 entreprises artisanales du BTP en France (source : CAPEB/FFB)
- ~90 000 entreprises de nettoyage/proprete
- ~50 000 entreprises de livraison/logistique TPE
- Filtre : celles avec 5-30 vehicules = environ 15-20% = **~75 000 - 100 000 cibles potentielles**
- Taux de conversion realiste : 1-2% = 750-2000 clients
- A 1.5 EUR/vehicule/mois x 15 vehicules en moyenne = ~22 EUR/mois/client
- **750 clients x 22 EUR = ~16 500 EUR MRR** (objectif atteignable en 12-18 mois)

---

## Risques identifies

1. **FleetNote est bien positionne** : concurrent direct francais, deja sur le creneau TPE. Differentiation par le prix et la simplicite extreme est possible mais pas evidente.
2. **Excel est "suffisant"** pour beaucoup : la migration vers un SaaS n'est pas urgente tant qu'il n'y a pas d'incident. Le declencheur est reactif, pas proactif.
3. **Le prix tres bas** (1-2 EUR/vehicule) implique un volume eleve de clients pour atteindre la rentabilite. Necessite une excellente strategie SEO/content.
4. **Pas de moat technique** : l'outil est simple a copier. La retention se fait par la data accumulee et la habitude.

---

## Verdict

**RETENUE** -- sous reserve de validation web approfondie sur le pricing de FleetNote et la taille reelle du segment "5-30 vehicules sans outil". Le positionnement "Excel killer ultra-cheap" pour les artisans avec quelques vehicules est un angle viable si le prix est suffisamment bas pour tuer l'hesitation. Le risque principal est la concurrence de FleetNote qui occupe deja ce creneau. A valider en phase de deep research.
