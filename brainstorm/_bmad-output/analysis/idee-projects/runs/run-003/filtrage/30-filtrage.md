## Suivi formations obligatoires pour TPE 1-10 salaries

## 1. CONCURRENCE (Score: 2/5 - Competition moderee a forte)

### Concurrents directs identifies par recherche web :

**MaSecuPro** - SaaS francais dedie QHSE pour TPE/PME
- Module specifique "Gestion des habilitations et formations"
- Automatisation du suivi des echeances de formation et de recyclage
- Generation automatisee de certificats de qualification signes electroniquement
- QR code securise pour chaque salarie = carte d'habilitation numerique
- Plan d'actions auto-genere pour les recyclages a venir
- **Positionne exactement sur le meme besoin** (suivi formations obligatoires TPE)
- Source : [MaSecuPro](https://www.masecupro.com/logiciel-habilitations)

**HRMAPS** - SIRH francais avec module formation
- Gestion de la formation "complete et puissante"
- Suivi des competences, planification, budgets formation
- Cible : PME a ETI (plutot 50+ salaries)
- Interface plus lourde, surdimensionnee pour TPE
- Source : [HRMAPS](https://www.hrmaps.fr/gestion-de-la-formation/)

**Empowill** - Logiciel RH specialiste parcours collaborateur
- Plan de formation, entretiens professionnels, GPEC
- Partenariat avec BRZ France pour le BTP
- Cible : PME et ETI, pas les micro-TPE
- Source : [Empowill](https://www.empowill.com/)

**Kelio** - Suite SIRH complete et modulaire
- Module formation avec personnalisation e-learning/blended learning
- Suite complete RH (recrutement, paie, temps, talents)
- Cible : PME 50+ salaries, trop lourd pour TPE
- Source : comparatifs Eurecia, Culture-RH

**Dokeos** - LMS specialise formations obligatoires/reglementaires
- Suivi rigoureux des conformites
- Oriente secteurs fortement reglementes
- Cible : organismes de formation et grandes entreprises
- Source : [Culture-RH - Logiciels gestion formation 2026](https://culture-rh.com/logiciel-rh/logiciel-gestion-formation/)

**Solutions internationales** (TalentLMS, iSpring, SkyPrep, Connecteam)
- LMS generiques avec suivi compliance
- Connecteam : gratuit pour petites entreprises (<30 users, 29 EUR/mois premium)
- Non adaptes au cadre reglementaire francais specifique (SST, CACES, HACCP...)
- Sources : comparatifs ComplianceQuest, EducateMe, Connecteam

**Passeport Prevention (outil gouvernemental)**
- Ouverture aux organismes de formation : 28 avril 2025
- Ouverture aux EMPLOYEURS : 16 mars 2026
- Obligation de declaration des formations sante/securite
- Gratuit, portail gouvernemental officiel
- **MENACE MAJEURE** : le gouvernement est en train de creer un outil gratuit qui couvre une partie significative du besoin
- Source : [Passeport Prevention](https://passeport-prevention.travail-emploi.gouv.fr/)

### Analyse concurrentielle :
**MaSecuPro est un concurrent direct quasi-parfait** pour les TPE/PME. Il couvre les formations obligatoires, les habilitations, les recyclages automatiques, avec QR codes. De plus, le **Passeport Prevention** (outil gouvernemental gratuit) ouvre aux employeurs en mars 2026 et couvrira le suivi des formations sante/securite. C'est la meme menace que OiRA/OPPBTP avait representee pour DUERP Express dans le run 1.

## 2. PREUVE DE DOULEUR (Score: 4/5 - Douleur forte et documentee)

### Recherches effectuees :
- "TPE formation obligatoire SST CACES oubli renouvellement probleme accident travail responsabilite penale"
- "passeport prevention formation obligatoire employeur TPE 2025 2026"

### Constats :
- **Douleur tres reelle** : les TPE n'ont pas de DRH, le gerant gere tout, les formations obligatoires sont nombreuses et variees selon le secteur (SST, CACES, HACCP, habilitations electriques, incendie, ATEX, travail en hauteur...)
- **Recyclages complexes** : SST tous les 24 mois, CACES validite 5 ans (mais actualisation recommandee), formations incendie annuelles...
- **Responsabilite penale confirmee** : en cas d'accident du travail sans formation adequate, le dirigeant engage sa responsabilite penale (homicide involontaire par imprudence)
- La CAPEB (artisans) et l'INRS publient des guides car les TPE sont perdues
- **Le Passeport Prevention** est cree JUSTEMENT parce que le gouvernement reconnait que les employeurs ne suivent pas les formations = validation de la douleur

### Sources :

## 4. DECLENCHEUR VERIFIABLE (Score: 3/5 - Obligations reelles mais pas de deadline urgente)

### Cadre legal confirme :
- **Code du Travail** : obligation generale de formation a la securite (articles L4141-1 et suivants)
- **SST** : recyclage obligatoire tous les 24 mois
- **CACES** : autorisation de conduite obligatoire (pas le CACES strictement, mais la formation adequate est requise)
- **HACCP** : obligatoire en restauration
- **Habilitations electriques** : normes NF C 18-510
- **Formation incendie** : obligatoire dans les ERP

### Sanctions confirmees :
- Responsabilite penale du dirigeant en cas d'accident (homicide involontaire)
- Amende 3 750 EUR par salarie en cas de defaut de formation
- Faute inexcusable de l'employeur (majoration rente, dommages et interets)
- Controles inspection du travail

### Pas de deadline specifique imminente :
- Les obligations existent depuis longtemps
- Le Passeport Prevention (mars 2026 pour employeurs) est un evenement mais c'est un OUTIL GRATUIT, pas une nouvelle obligation
- Pas d'effet "vague reglementaire" comparable a la loi Le Meur ou la loi influenceurs

### Sources :
- [Secours Prevention - Formations obligatoires](https://secours-prevention.com/les-formations-en-securite-obligatoires-en-entreprise/)
- [CACES-Permis.fr - Renouvellement obligation](https://www.caces-permis.fr/renouvellement-caces-obligation-employeur-ce-que-dit-la-loi/)

## VERDICT : ELIMINEE

### Raison principale :
**Double menace concurrentielle fatale.**

1. **MaSecuPro** est un concurrent direct parfaitement positionne sur les TPE/PME, avec exactement les fonctionnalites visees (suivi habilitations, recyclages automatiques, certificats QR code). Il existe, il fonctionne, il est deja sur le marche.

2. **Passeport Prevention** : le gouvernement ouvre un outil GRATUIT aux employeurs le 16 mars 2026 pour suivre les formations sante/securite de leurs salaries. C'est EXACTEMENT le pattern qui a tue DUERP Express dans le run 1 (outil OiRA/OPPBTP gratuit). Le Passeport Prevention va devenir la reference obligatoire, rendant un SaaS prive redondant sur la partie core du besoin.

### Lecon du run 1 qui se repete :
> "Government free tools are a major competitive threat" (MEMORY.md)

Le Passeport Prevention est la version formation du DUERP OiRA. Meme si l'outil gouvernemental sera basique, il sera gratuit et obligatoire, ce qui tue le marche pour un outil prive sur le meme besoin.

### Note :
La taille de marche (908K TPE) est tentante, mais un grand marche avec un concurrent gratuit gouvernemental + un SaaS dedie existant = piege a eviter.
