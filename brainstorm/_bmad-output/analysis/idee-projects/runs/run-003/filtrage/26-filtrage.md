### Analyse concurrentielle

**Points cles :**
- **ALERTE : Trackforce Valiant a leve 150M$+** = critere d'elimination potentiel
  - MAIS : Trackforce cible les GRANDES structures (30 000 sites clients dans 50 pays). Pas les TPE francaises de 5-50 agents. Le critere 100M$ s'applique au meme segment.
- **Sekur est le vrai concurrent** : SaaS francais specialise securite privee, article dedie "Audit de Conformite CNAPS 2026", forte presence SEO
  - Cependant, Sekur est un logiciel de GESTION COMPLETE (planning, facturation, RH), pas un outil leger CNAPS-only
- Comete est un autre acteur francais specialise securite privee
- **Aucun outil leger "CNAPS compliance checker"** identifie : tous les concurrents sont des ERP/logiciels de gestion lourds

**Verdict concurrence : Le marche a un gorille (Trackforce 150M$) MAIS sur un segment different (grands comptes internationaux). Sur le segment TPE francaise, Sekur et Comete sont les principaux concurrents. Le creneau "compliance CNAPS-only pour TPE" reste ouvert.**

- La croissance du secteur (+21% CA TPE) est un point positif

## VERDICT : RETENUE (avec reserves majeures)

### Raisons de retention
- Douleur extreme et quantifiable (3.9M EUR sanctions, +58%)
- Niche specifique non adressee par les concurrents (CNAPS-only pour TPE)
- Willingness to pay elevee (les amendes font tres peur)
- Trackforce est hors segment (grands comptes internationaux)
- Croissance du secteur (+21% CA TPE)

### Reserves majeures
1. **Marche etroit** : 3 000-6 500 cibles est petit. Le revenu par client devra etre eleve (50-100 EUR/mois minimum) pour atteindre un CA significatif.
2. **Sekur est bien installe** : meme s'il est "lourd", il pourrait facilement ajouter un module CNAPS compliance simplifie
3. **Pas de deadline unique** : plus dur a marketer qu'AccessiBoard. Le message devra etre "preparez votre prochain audit" plutot que "la deadline arrive le X"
4. **Risque d'adjacence** : Sekur ou Comete pourraient copier la fonctionnalite en quelques mois
